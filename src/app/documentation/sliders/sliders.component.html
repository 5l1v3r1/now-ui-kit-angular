Sliders

Install

npm i --save nouislider ng2-nouislider

Import

import {{'{'}} NouisliderModule {{'}'}} from 'ng2-nouislider';

For more informations, see documentation here. 
In HTML:
<nouislider class="slider slider-danger" [min]="0" [max]="10000" [step]="10" [(ngModel)]="simpleSlider" [tooltips]="true"></nouislider>
<br/>
<nouislider class="slider slider-info" [connect]="true" [min]="0" [max]="10000" [step]="10" [(ngModel)]="doubleSlider" [tooltips]="true"></nouislider>

In Typescript:
doubleSlider = [1000, 5000];
simpleSlider = 3000;
